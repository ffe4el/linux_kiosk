컴파일을 수행하기 위한 client 작성 스크립트

<컴파일>
(server) gcc -o server server.c
(client) gcc -o client client.c

<실행시작>
1. ./server 로 터미널에서 서버코드 실행 
2. server터미널에서 바로 판매모드를 진행할것인지(1 입력), 관리자모드로 들어갈 것(2 입력)인지 결정
3. 2번 Manager mode를 선택하면 비밀번호(비번 : 1234)를 입력해야함. 만약 비밀번호가 일치하면 관리자모드로 들어오게됨
4. 관리자모드로 들어오게 된다면 음식리스트를 설정할 수 있음. 등록할 음식의 갯수를 입력함 (예 : 3)
5. 3개의 음식을 이름, 가격, 수량 순으로 등록 (각각 띄어쓰기를 통해 입력! 예 : coke 2000 30)
6. 음식을 다 등록하게 되면 클라이언트와 연결할 수 있는 상태가 됨
7. client 터미널로 가서 ./client로 클라이언트 코드 실행
8. 만약 위에 2번에서 1번 판매모드를 선택하게 되면 바로 client 터미널에서 ./client로 클라이언트 코드 실행
<client가 연결된 후>
9. client코드 에 들어오고 나면 2분의 시간제한이 걸려있음. 그동안 영화티켓과 음식을 고르고 계산까지 진행해야함
10. 구매를 진행하고 싶다면 1 을 누른다.
11. 영화 목록과 티켓 가격이 쭉 뜬다. 여기서 보고싶은 영화의 인덱스를 입력하면 된다. (예 : 1)
12. 해당 영화의 구매 가능한 티켓 개수가 뜨고 본인이 몇개의 티켓을 구입하고 싶은지 입력한다.(예 : 2)
13. 만약 남은 티켓개수가 입력한 티켓 개수보다 많다면 available이라고 뜨고 나이를 입력할 수 있다.(예 : 22 23)
14. 총 금액이 나오고 좌석을 입력한 티켓 수대로 선택할 수 있다. (예 : 1 1)
15. 좌석을 다 고르면 푸드코트에서 음식을 살 수 있다. 음식리스트가 나오고 원하는 음식의 인덱스를 입력한다. (예 : 1)
16. 원하는 수량도 적는다. (예 : 2)
17. 다른 종류의 음식을 고르고 싶으면 해당 음식의 인덱스를 또 적을 수 있고, 그렇지 않다면 계산 단계로 넘어갈 수 있다.(계산모드로 가려면 0 입력)
18. 영화값과 음식값이 합쳐진 총 금액이 출력되고 원하는 만큼 금액을 나누어서 지불할 수 있다. (예 : 2000 3000 4000)
19. 총 금액이 모두 지불되면 클라이언트는 종료된다.
